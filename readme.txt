Delivers contacts and leads directly to your PipeDrive CRM pipeline.

== Description ==

This plugin adds People directly into your PipeDrive crm. 

== Installation ==

1. Upload `wptopd` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Drag the plugin to your sidebar to activate.
